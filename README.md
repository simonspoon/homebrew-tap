# homebrew-tap

Homebrew formulae for Simon's CLI tools.

## Usage

```bash
brew tap simonspoon/tap
brew install <formula>
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `helios` | Tree-sitter code indexing CLI with SQLite storage |
| `khora` | Web app QA automation CLI via Chrome DevTools Protocol |
| `limbo` | CLI task manager designed for LLMs and AI agents |
| `loki` | Desktop app automation CLI for macOS QA testing |
| `nyx` | Index and search Claude Code conversation history |
| `qorvex` | iOS Simulator and device automation toolkit for macOS |
| `suda` | Structured memory and knowledge management CLI for AI agent workflows |
| `wisp-cli` | CLI companion for Wisp, a desktop UI design tool for agents |
